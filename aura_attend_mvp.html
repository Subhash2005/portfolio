Aura Attend – MVP

Project Overview

Aura Attend is a smart attendance and student management platform developed to simplify attendance tracking, leave management, OD submission, and real-time monitoring for educational institutions and organizations.

Core Features

Attendance Management

  • Real-time attendance posting
  • Attendance tracking and history
  • Subject-wise attendance monitoring
  • Automated attendance updates

Leave and OD Management

  • Online leave application
  • OD submission and approval workflow
  • Permission request management
  • Leave status tracking

Payment Management

  • Leave fee payment integration
  • Payment status monitoring
  • Receipt and transaction tracking

Admin Dashboard

  • Attendance analytics and reports
  • Student activity monitoring
  • Present and absent statistics
  • Department-wise data management

Cloud Storage and Media Handling

  • Cloudinary integration for storing videos and images
  • Optimized media delivery and management

Authentication and Database

  • Secure JWT-based authentication
  • MongoDB cloud database integration
  • Role-based access control
